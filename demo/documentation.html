Example : Censor Single Word

Code :

<div class="actress"> <h3>Actress</h3> <ul> <li>Nicole Kidman</li> <li>Kristen Stewart</li> <li>Jodie Foster</li> <li>Catherine Zeta Jones</li> </ul> </div> <script> $(".actress").censor({ badWords : 'Kristen Stewart' }); </script>

Output :

Actress

  • Nicole Kidman
  • Kristen Stewart
  • Jodie Foster
  • Catherine Zeta Jones

Example : Censor Multple Words

Code :

<div class="actress"> <h3>Movies</h3> <ul> <li>Godfather I</li> <li>Pirates Of The Caribian</li> <li>Twilight</li> <li>Mission Impossible</li> <li>Vampire</li> </ul> </div> <script> $(".movie").censor({ badWords : ['vampire', 'twilight', 'aaa'] }); </script>

Output :

Movies

  • Godfather I
  • Pirates Of The Caribian
  • Twilight
  • Mission Impossible
  • Vampire

Example : Censor with your custom censored word

Code :

<div class="singer"> <h3>Singers</h3> <p>Michael Jackson, Bryan Adams, Justin Bieber, Dido</p> </div> <script> $(".singer").censor({ badWords : 'Justin Bieber', replaceWord : '<span style="color:#ff0000;">--CENSORED--</span>' }); </script>

Output :

Singers

Michael Jackson, Bryan Adams, Justin Bieber, Dido
